# Solvorr Agency OS - Sales Demo

A premium, interactive internal operations dashboard designed for marketing agencies.

## Features

- **Dashboard Overview**: Real-time KPI tracking, revenue trends, and activity feeds.
- **CRM & Client Management**: Full client database with status tracking and detail views.
- **Sales Pipeline**: Kanban-style deal management with drag-and-drop simulation.
- **Revenue & Payments**: Invoice tracking, service-based revenue breakdown, and payment status management.
- **Campaign Operations**: Task management system with workload visualization.
- **Automation Engine**: Interactive simulation of inbound lead workflows.
- **Team Management**: Workload distribution and role-based member cards.
- **Public Lead Form**: Polished external-facing lead capture page.

## Tech Stack

- **React + Vite**
- **TypeScript**
- **Tailwind CSS** (v4)
- **shadcn/ui**
- **Lucide React** (Icons)
- **Recharts** (Data Visualization)
- **Framer Motion** (Animations)
- **Zustand** (State Management)

## How to use the Demo

1. **Login**: Use the default credentials to enter the dashboard.
2. **Explore CRM**: View and edit client details in the CRM tab.
3. **Manage Pipeline**: Move deals through stages in the Pipeline view.
4. **Simulate Automation**: Go to the **Automation** page and click "Simulate New Lead Flow" to see the engine in action.
5. **Public Form**: Visit `/apply` to see the external lead capture page. Submitting the form will automatically trigger the automation flow in the dashboard.
